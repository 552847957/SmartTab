Settings:
keyNavigation(true) : Use left and right arrow keys to navigate tab.
autoProgress(true) : Tabs automaticaly navigate with intervel specified(default:- progressInterval:3500).
stopOnFocus(true) : Stops autoProgress when mouse hover on control and restarts when not.
TransitionEffect(slide) : SlideUp/SlideDown effect on tab transition
